# 基本功能

下面介绍一些 LCD 的基本功能及其注意事项。

## 路线名称的绘制

LCD 上常常需要绘制路线名称，例如左上角的当前路线和各个车站的换乘路线等。

为了拟真，LCD 支持用路线号码代替路线名称进行绘制。如果一个路线定义了路线号码，则 LCD 将始终绘制路线号码的内容；否则，LCD 仍将绘制路线名称，但不包括 `||` 之后的字符。

::: tip 最佳实践
由于路线号码不支持多语言，建议仅对数字形式名称的线路定义路线号码，而对于文字形式名称的线路不定义。例如：

| 路线名称  | 路线号码 |
| ------------- | ------------- |
| `4 号线\|Line 4`  | `4`  |
| `大兴线\|Daxing Line`  | (不定义)  |

:::

::: warning 提醒
取消路线号码时，不能仅仅取消勾选“有路线号码”复选框，必须将“路线号码”文本框中的字符清除。
:::

## 换乘信息的绘制

获取换乘信息时，LCD 遵循以下规则：

- 同样的换乘信息仅绘制一次。

- 对于跨线运营列车，不绘制列车下一条线路的换乘信息。

  以北京地铁的四惠和四惠东站为例，它们都是 1 号线和八通线的换乘站。1 号线列车的 LCD 将显示这两个车站均可换乘八通线；但对于 1 号线和八通线贯通运营的列车，这两个车站将不显示换乘信息。

- 如果某个车站有与其连接的车站，以空心矩形绘制连接车站的换乘信息。
  
  在 MTR 中，如果两个车站的范围有重叠，就说它们相互连接。绘制某个车站的换乘信息时，对于与其连接的车站的换乘信息，LCD 将以空心矩形绘制。

## 环线

目前不支持环线的显示，因此对于环线列车的 LCD，某些信息可能不准确。
